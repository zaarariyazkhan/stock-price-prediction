# Stock Price Prediction using Machine Learning

Predicting closing prices of NSE stocks using ML models.

## Models Used
- Linear Regression
- Random Forest
- LSTM
- GRU

## Dataset
- Reliance Industries (RELIANCE.NS) — 2019 to 2024
- Validated on TCS.NS
